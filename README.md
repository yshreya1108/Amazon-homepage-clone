# Amazon Homepage Clone

A responsive Amazon homepage clone built with **HTML** and **CSS**, designed to replicate the look and feel of Amazon’s main landing page. This project focuses on front‑end development fundamentals, responsive design, and layout structuring.

---

## 🚀 Features
- Sticky **navbar** with logo, search bar, and cart
- Responsive **hero section** with centered message
- Adaptive **panel** for categories and deals
- Product **grid layout** with flexible boxes
- Multi‑column **footer** with service links and social media
- Fully responsive across **desktop, tablet, and mobile**

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling and responsiveness
- **Media Queries** for adaptive layouts

---

## 📱 Responsiveness
- Desktop → full layout with 4 product boxes per row
- Tablet → 2 product boxes per row, adjusted navbar
- Mobile → stacked layout, collapsible navbar, scrollable panel
